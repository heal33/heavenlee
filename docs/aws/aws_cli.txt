aws command examples

List contents of an S3 bucket
aws s3 ls s3://bucket-name

List contents of an S3 bucket directory
aws s3 ls s3://bucket-name/directory/

Upload a File to S3
aws s3 cp local-file.zip s3://bucket-name/folder/remote-file.zip

Delete a file from S3
aws s3 rm s3://bucket-name/directory/profile.txt
